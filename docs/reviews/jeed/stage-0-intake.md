# JEED 阶段零：快速初筛

检查对象：`Just Enough Effects Descriptions 1.21-2.3.2`

## 用途

为 JEI、REI 与 EMI 增加状态效果说明页，展示效果的作用、来源和相关物品；在背包等带有效果图标的界面中，也可提供点击和悬浮提示。

## 初判

- 物品：`不适用`。JAR 没有物品模型、方块、实体、战利品表或世界生成资源；药水图标只用于 Catalogue 展示。
- 服务端：`client_only`，高置信度。所有 Mixin 声明在 `client` 数组，运行期只订阅客户端 `ScreenEvent`；没有网络包、服务端 tick、存档或世界内容。
- 汉化：`需要处理`。已有原版中文基础，但需按当前整合包补齐缺少的效果说明。
- Catalogue：`需要处理`。应提供双语用途说明、仓库页面和预填 JEED 问题标题。

后续阶段按该初判快速通过服务端与物品等级，只审查代码、汉化和 Catalogue 条目。
